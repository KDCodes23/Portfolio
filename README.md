# Kanishka Dutta Portfolio

A modern personal portfolio website built to showcase my work as a developer, designer, and founder of Vertex Studio Canada.

This portfolio is designed to be more than a simple resume page. It highlights my projects, technical skills, experience, and creative work in one place. The goal is to give recruiters, collaborators, and clients a clear view of what I can build and how I approach software development.

Live site: (https://kdcodes.dev/)

---

## About This Project

This is my first portfolio repo that I consider resume-ready. The site is currently live and will continue to evolve as I improve the design, add stronger project case studies, polish animations, and refine the content.

The portfolio focuses on:

- Showcasing real projects and technical experience
- Presenting my skills in frontend and full-stack development
- Highlighting Vertex Studio Canada and my role as founder
- Creating a clean, responsive, and professional online presence
- Building a long-term portfolio that can grow with my career

---

## Tech Stack

- Next.js 15 with App Router
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Lenis smooth scrolling

---

## Features

- Responsive design for desktop, tablet, and mobile
- Smooth scrolling experience
- Animated sections and transitions
- Project showcase section
- Skills and tech stack section
- Experience timeline
- Vertex Studio Canada section
- Contact section for opportunities and collaborations
